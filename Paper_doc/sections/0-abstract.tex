\begin{abstract}
We address the growing communication bottleneck in synchronous federated and distributed deep learning, intensified by ever-growing model size. We present a learned distributed source coding (DSC) method that quantizes worker updates using a learned successive Wyner--Ziv (WZ) scheme. The server uses recent reconstructions across all workers as side information, converting cross-worker and temporal redundancy into bit-rate savings while preserving model quality.
Our approach is grounded in Wyner--Ziv limits for correlated sources, motivating coding near conditional rates when side information is available. In contrast to past studies relying on uniform quantization or sparsification, we introduce: (i) a multi-terminal per-coordinate learned WZ quantizer with refinement stages, (ii) a protocol applying said quantizer in standard FL training.
Our findings position learned inter-worker DSC quantization as a scalable communication primitive for federated learning and large-scale distributed training.
\end{abstract}
