We present our framework exploiting multi-terminal approach on correlated gradients. This correlation is observed in prior works~\cite{ZhangTao2022, Abrahamyan2022LGC} and confirmed for per-coordinate correlation coefficients in~\ref{fig:grad_corr}.
The distribution shift was investigated and shown to be small assuming scale normalization.
These enable the use of DSC methods, like a WZ quantizer.

\subsection{Learned Successive Multi-terminal Wyner--Ziv Quantization (WZQ)}
\label{subsec:method-lsmtwzq}
At round \(r\), worker \(k\in\{1,\ldots,K\}\) holds flattened gradient \(G_{k,r}\in\mathbb{R}^n\), preprocessed to \(X_{k,r}\). Each element is quantized to symbols \(u_{k,r,i,j}\) (spatial index \(i\), refinement stage \(j\)). Symbol lists \(U_{k,r,j}\) are further compressed to bitstreams \(O_{k,r}\) using SW coder paired with learned conditional prior estimating \(p(u_{k,r}|{Y}_{k,r})\).

Server decodes using SW decoder and side information (\textbf{SI}) to recover \(U_{k,r,j}\), then dequantized using the SI into \(\hat X_{k,r}\) to finally be post-processed into \(\hat G_{k,r}\). Figure~\ref{fig:block} illustrates the process.

\textbf{Pre-/Post-processing:} Layer normalization aids training stability as some layers have very small scale. Outliers are separately encoded as they are few in number.

\subsubsection{Training Quantizers and SI variants}
Each worker has independent quantizer instances. Decoder transmission is negligible (100KBs decoder vs 11 million parameters in resnet-18).
We consider two variants of the quantization process which are defined by where the quantizer is trained and what is included in the SI.

\textbf{All-SI quantization process:}
Our main process as it exploits inter-node correlation.
Trained on the server, it uses all workers' past reconstructions as SI except for one; the most recent reconstruction (for the worker in question) is excluded and used as the training target. This leverages both inter-node and temporal correlation but introduces a train--inference distribution mismatch, which we observe is usually the dominant cause of higher rate--distortion cost. Another issue is that the resulting higher distortions degrade the reconstructions used on the next training phases, leading to multi-round error accumulation---an issue we address with the next quantization process.

\textbf{Temporal-only quantization process}
SI consists only of the same worker's past reconstructions produced by this quantization process. Training occurs on the worker side, so the true gradient to be transmitted can be used as the training target, minimizing train--inference mismatch. Because the decoder is available at both ends, the server and worker can maintain an identical list of reconstructions. The benefit of low train--inference mismatch comes at the cost of exploiting only temporal (not inter-node) correlation, so this variant is used periodically to realign reconstructions to the true gradients and mitigate error accumulation. In our experiments, it is very powerful, but we do not pursue it further because it cannot exploit inter-node correlation.

\subsection{Proposed Protocol leveraging WZQ}
\label{subsec:method-protocols}
To maximize inter-node redundancy via the all-SI quantization process, we use the five-phase protocol.: \emph{Initialization}, \emph{Warm-up}, \emph{Realignment}, \emph{Retraining}, and \emph{Freeze}. Fig.~\ref{fig:protocols_timeline} shows the overall protocol.

\noindent\textbf{1) Initialization.}
Round 1 uses a pre-trained non-WZQ quantizer (no SI), trained on a Laplacian source without SI; its decoder is sent to workers so each can form a first local reconstruction, used by both temporal-only and all-SI.


\noindent\textbf{2) Warm-up.}
With one reconstruction per worker available, train an all-SI quantizer progressively: in round 2, training for agent 2 uses agent 1's fresh reconstruction as SI; continue for \(S-3\) rounds, where \(S\) is the max number of stored reconstructions per worker.

\noindent\textbf{3) Realignment.}
Obtain two high-quality reconstructions---one to augment the SI pool and one to serve as the training target for all-SI in retraining. While high-rate all-SI could produce these, at high rates temporal-only shows better rate--distortion; thus, per round, train a worker-side temporal-only quantizer in parallel and use it.

\noindent\textbf{4) Retraining.}
Using these high-quality (but high-rate) reconstructions, train an all-SI quantizer for all workers in parallel on the server.

\noindent\textbf{5) Freeze.}
Freeze the retrained quantizer and the SI list used to train it, and use this fixed pair for the next \(S-3\) rounds; phases 3--5 then repeat thereafter.
