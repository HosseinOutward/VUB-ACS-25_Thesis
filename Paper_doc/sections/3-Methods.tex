\subsection{Simple Framework}
\label{ssec:simpleframework}
The simple framework consists of the following components:
a) preprocessing, b) warmup phase, c) a single main training phase, and d) use frozen model for all downstream rounds

\figure[ht]
    \centering
%    \includegraphics[width=0.5\textwidth]{figures/simple_framework.png}
    \caption{Simple Framework}
    \label{fig:simple_framework}
    The figure shows the simple framework for training the model. The model has 3 phases: a warmup phase and a main training phase and finally a frozen phase. In the warmup phase, we use a pretrained model to get initial side information. In the main training phase, we train the model with the data we have seen so far. Finally, in the frozen phase, we use the trained model for all downstream rounds and only change the side information based on the latest received gradients.
\endfigure
