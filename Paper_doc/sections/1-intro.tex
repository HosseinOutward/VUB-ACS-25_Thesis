\section{\textbf{Introduction}}
\label{sec:intro}

Federated learning enables collaborative model training on decentralized data by exchanging model updates like gradients. However even with compression techniques such as quantization and sparsification, communication costs stay a significant overhead, remaining a primary bottleneck for distributed learning scalability.

More advanced methods improve rate by exploiting the link between gradients across time and nodes, reducing redundant transmissions.


Distributed Source Coding (DSC), based on Slepian-Wolf \cite{SlepianWolf1973} and Wyner-Ziv \cite{1055508} principles, enables workers to independently compress correlated gradients while the parameter server acts as joint decoder, using other workers' gradients as side information. This exploits inter-node redundancy to reduce communication rate to joint entropy boundry without direct worker communication.

Conventional DSC methods rely on rigid algorithms unsuited for large-scale gradients compression. Recent works \cite{OzyilkanRecoverBinning} propose learned Wyner-Ziv quantizers for Gaussian sources which are adaptable to gradient distributions and multi-terminal settings. They train end-to-end models with learned non-uniform quantizers, and dequantizers that jointly process codes and side information for lower rates.

This framework extends to successive refinement \cite{JoukovskyLayeredWZ}, sending base layers followed by optional residual layers, offering better rate-distortion tradeoffs and bandwidth adaptation flexibility.

In multi-terminal scenarios, multiple side information aids decoding. The server sequentially decodes updates from each agent, adding them to side information lists, enabling progressively reduced rates for subsequent workers.

We present a unified framework combining federated learning with learned successively refined Wyner-Ziv quantizers in multi-terminal settings. Motivated by proven DSC bandwidth savings in FL \cite{ZhangTao2022, Abrahamyan2022LGC} and learned WZ quantizer gains, we introduce a fully learned, multi-terminal, end-to-end pipeline for gradient compression with focus on inter-node redundancy.
