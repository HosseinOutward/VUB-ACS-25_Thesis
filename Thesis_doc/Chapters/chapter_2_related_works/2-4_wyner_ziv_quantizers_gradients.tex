\section{Learned Layered Coding for Successive Refinement in the Wyner-Ziv Problem \cite{JoukovskyLayeredWZ}}
This paper extends the work done in \cite{OzyilkanRecoverBinning} which propose a monolithic, single stage wz quantizer, by proposing a layered coding scheme for the Wyner-Ziv problem. The main idea is to have multiple encoders and decoders, working in a successive refinement manner. Each stage will give a seperate quantized output, which will be used in the next encoding stage; with this method one would have mutliple vectors of quantized values which could be further compressed using a DSC entropy coder. The bitrate calculation for this method is different from the one used in \cite{OzyilkanRecoverBinning}, as it sums the rate of each stage separately and improves on the rate distortion trade off as seen from the figure \ref{fig:layered_wz_imporvments} below. The architecture of the layered WZ quantizer is shown in figure \ref{fig:layered_wz_architecture} below.
