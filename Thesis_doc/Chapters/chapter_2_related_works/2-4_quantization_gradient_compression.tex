\section{Wyner–Ziv Gradient Compression for Federated Learning}
This work \cite{liang2021wynerziv} is similar to our "Worker-Side" protocol proposed in Chapter 3, as it uses historical gradients as side information for a WZ quantizer.
This work exploits temporal correlation between consecutive rounds by treating the server's historical gradients as decoder-side side information and designing a practical Wyner–Ziv–style quantizer. Concretely, they build a modulo quantizer $Q$ that encodes each gradient parameter into $\log_2 s$ bits; the server reconstructs using the side-information value, achieving unbiasedness and bounded error without assuming a particular gradient distribution. They integrate this into LQSGD and report better communication–accuracy trade-offs than QSGD on linear models and ResNet-18/CIFAR tasks.
