\section{Data, Model, and Training Setting}

The configuration is SVHN with IID split, trained with resnet18 in a synchronous FL setup with SGD optimizer.

The grads are accumulated per batch in each workers memory and at the end it will be sent to the server using a compression protocol which is supposed to compress at the worker and reconstruct the data at the server

The reconstructed grads will be given to fedSGD to compute an average grad and applying it to the model as part of the aggregation step. the resulting model is then sent to workers, potentially using the similar protocols.
