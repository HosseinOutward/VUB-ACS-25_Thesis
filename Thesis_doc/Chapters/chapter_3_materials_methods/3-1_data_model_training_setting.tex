\section{Data, Model, and Training Setting}

The primary experimental configuration uses the SVHN dataset with an IID split across workers, training a ResNet-18 model in a synchronous federated learning (FL) setup with the SGD optimizer.

Each worker accumulates per-batch gradients in local memory and, at the end of the round, transmits them to the server via one of the developed compression protocols. Each protocol performs gradient pre-processing, compression, and transmission; on the server, the corresponding decoder handles decompression and post-processing to obtain the reconstructed gradients. FedSGD then averages the reconstructed gradients and applies the resulting update to the global model during aggregation. The updated model is subsequently broadcast to the workers, potentially using the same or analogous protocols.
